### 主要集合类型
- list 有序，可以重复
- set 不重复
- queue 支持先入先出，后入先出等


#### set集合类型
- TreeSet 支持自然顺序访问，添加删除等操作低效
- HashSet 利用哈希，常数时间的添加，删除，包含操作，不保证有序
- LinkedHashSet 额外构建一个记录插入顺序的双向链表。


#### java 9提供了List.of() 和 set.of()
- 简化代码量
- 保证不可变性，线程安全，空间紧凑，不需要扩容
